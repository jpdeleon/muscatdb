prose Photometry Pipeline Workflow

Detailed step-by-step reduction flow of the calibrated science frames within the prose photometry engine.

▼ 
graph TD
    %% Input
    A["Raw Science FITS Files\n(Calibrated e91.fits)"]:::default --> B["Band Grouping\n(Grouped by gp, rp, ip, zs filters)"]:::default
    
    %% Setup & Reference Frame
    B --> C["Reference Frame Creation\n(Median stack of middle frame index)"]:::default
    C --> D["Point Source Detection\n(Brightest stars on reference frame)"]:::default
    D --> E["Celestial Target Identification\n(WCS matching / MAST coordinate resolution)"]:::default
    E --> F["Gaia Contamination Sizing\n(Sizes sky annulus RIN & ROUT to exclude contaminants)"]:::default
    F --> G["Aperture Sizing Grid\n(Multiple radii defined from FWHM up to RIN)"]:::default
    
    %% Frame-by-Frame Parallel Photometry
    B --> H["Parallel Frame Reduction\n(prose.SequenceParallel)"]:::pipeline
    G --> H
    H --> H1["CCD Trim\n(Trim borders/edge noise)"]:::default
    H1 --> H2["Gaussian Modeling\n(Gaussian2D fit for FWHM)"]:::default
    H2 --> H3["Twirl Alignment\n(ComputeTransformTwirl & AlignReferenceSources)"]:::default
    H3 --> H4["Centroid Tuning\n(CentroidQuadratic optimization)"]:::default
    H4 --> H5["Flux Extraction\n(AperturePhotometry & AnnulusBackground)"]:::default
    
    %% Post-processing
    H5 --> I["Differential Photometry\n(autodiff - Broeg et al. 2005)"]:::default
    I --> J["BJD Time Correction\n(Converts JD-UTC to BJD-TDB via barycorrpy)"]:::default
    J --> K["Export Results\n(CSV files, apertures.png, alignment.png, gif)"]:::default

Pipeline Execution Details

1 FITS Header Scanning
Extracts metadata from raw scientific FITS files. It excludes calibration frames (dark/flat/bias) and writes clean logs.
muscat-db scan <inst> <yymmdd>
2 Database Assembly
Ingests the scanned CSV files, building indices and tables in a unified, optimized SQLite database.
muscat-db build-db
3 Interactive Browsing
Allows real-time target searches (with regex support), logs inspection, and observation schedules directly from the Web GUI.
muscat-db serve
4 Photometry Reduction
Launches a calibrated background reduction using the prose pipeline to align frames, build PSF models, and extract raw lightcurves.
5 Transit Fitting
Runs a PyMC MCMC sampler to model the transit curve, compute planetary/stellar parameters, and output high-quality fit plots.
6 Job Monitoring
Tracks active, pending, or completed photometry and transit fitting background processes. Shows logs and error tracebacks.
