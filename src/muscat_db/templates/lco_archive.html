Search by Request ID

Enter the numeric request ID from an observation portal URL (e.g. 4236675 from observe.lco.global/requests/4236675) to pull every frame for that request. It ignores the search fields above; results appear below for selection and download.

Search LCO Archive

By default the Object name is resolved to sky coordinates (NASA/TOI catalog, then SIMBAD) and every frame whose footprint covers that position is returned — robust to naming variants (WASP-12 / Wasp-12 / WASP12). Enable Fuzzy name match to fall back to matching the raw OBJECT header text instead.

Files save to <MUSCAT_LCO_DIR>/<inferred instrument>/<date>/ (or that instrument's data dir). The destination instrument is inferred from the archive metadata: OGG 2m0 MuSCAT → muscat3, COJ 2m0 MuSCAT → muscat4, and LCO 1m0 → sinistro.
